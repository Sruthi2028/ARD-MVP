## Administrative Reality Drift (ARD)

Administrative Reality Drift (ARD) is a simple governance metric that measures the gap between officially announced administrative plans and actual on-ground execution.

## Problem
Large public development projects often suffer from delays, policy changes, and execution inefficiencies. There is no easy way to quantify how much administration has drifted from its original plans.
